# 배포 안내

주 사용자: 같이

## 이 문서는 무엇인가요?

이 문서는 게임을 인터넷에 올릴 때 보는 문서입니다.

사용자와 AI가 배포 방법을 같이 확인할 때 사용합니다.

## 지금 배포하는 곳

- GitHub Pages

## 배포 주소

- `https://gabrielwithappy.github.io/castle-depense/`

## 배포할 때 하는 일

### 1. 게임이 잘 실행되는지 확인합니다

- `index.html`이 열리는지 확인
- 중요한 버튼이 보이는지 확인
- 게임이 멈추지 않는지 확인

### 2. 필요한 파일이 준비되었는지 확인합니다

- 메인 실행 파일
- 게임 코드 파일
- 화면에 필요한 리소스

### 3. Git에 올립니다

```bash
git add -A
git commit -m "Update game"
git push origin main
```

### 4. 배포 후 확인합니다

- 사이트 주소가 열리는지 확인
- 모바일에서도 보이는지 확인
- 예전 화면이 보이면 새로고침합니다

## 주의할 점

- 경로가 잘못되면 화면이 안 나올 수 있습니다.
- 캐시 때문에 이전 버전이 보일 수 있습니다.
- 배포 직후에는 잠깐 404가 나올 수도 있습니다.

- 배포 주소를 확인할 때
- 배포 방식이 바뀔 때
